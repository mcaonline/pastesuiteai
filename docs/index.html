Welcome

Everything you need to know about PasteSuiteAI.

PasteSuiteAI is a Windows desktop tool for smart text actions, AI-powered text transforms, and speech-to-text transcription. It works with your own API keys — bring a key from OpenAI, Anthropic, Google Gemini, Mistral, Azure OpenAI, or connect to a local model via Ollama or another compatible endpoint. Your keys are stored securely in your OS credential store and never leave your machine.

Key Features

  • Paste as Plain Text — strip all formatting from copied text with a single hotkey.
  • AI Text Transforms — rewrite, translate, correct, summarize, or apply any custom LLM prompt to selected text.
  • Speech to Text — record audio from your microphone and transcribe or translate it using a cloud or local STT provider.
  • PromptBar — a global prompt input for free-form AI questions, plus one-click quick prompt buttons.
  • Prompt Library — save, organize, and instantly recall prompts from an overlay that works in any application.
  • Action Chaining — feed the output of one action into the next to build multi-step processing pipelines.
  • Result Pop-Up Window — display AI output in a floating overlay near your cursor, without leaving the current application.
  • Custom API Pipelines — connect to any REST API with multi-step request sequences, loops, and conditionals.
  • History — browse recent entries and action results within your session.

Quick Start

New to PasteSuiteAI? Follow these steps to get up and running:

  1. Install & set up — download the installer, run the Setup Wizard, and connect your first AI provider.
  2. Explore Actions — enable the built-in actions or create your own custom text transforms.
  3. Configure Connections — add more providers, tune system prompts, and set capability routing defaults.
